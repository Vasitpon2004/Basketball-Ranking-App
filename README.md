# Basketball-Ranking-App
Full-stack web application for Basketball league management and ranking.
---
# Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
---
# Prerequisites
ก่อนติดตั้ง กรุณาตรวจสอบให้แน่ใจว่าในเครื่องของคุณมีโปรแกรมเหล่านี้:
- [Node.js](https://nodejs.org/) (เวอร์ชัน 18 ขึ้นไป)
- [PostgreSQL](https://www.postgresql.org/) (ติดตั้งในเครื่อง หรือใช้ Cloud Database)
---
